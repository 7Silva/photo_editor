# Photo Editor

A simple script to edit photos in an automated way.

## Features

 - [x] Add text in image;
 - [ ] Add filter in image;
 - [ ] Create image with text;

## Dependencies
- Chalk;
- Sharp;
- Node.js;
- Inquirer;
- TypeScript;
    - @types/inquirer;
    - @types/sharp;

## Running
1. Run `yarn or npm install`;
2. Run `yarn build`;
3. Run `yarn start`;
